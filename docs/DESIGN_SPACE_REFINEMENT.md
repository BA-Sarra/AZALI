# AZALI layout spacing refinement

This update makes the public storefront calmer on desktop screens.

Changes included:

- Reduced public product and category card sizes.
- Product grids now use four columns on large desktop screens so each card is easier on the eyes.
- Category cards use shorter image ratios and smaller typography.
- Product cards use smaller headings, smaller swatches, and lighter padding.
- The homepage COD/payment information card was removed.
- The homepage philosophy block is now a quiet editorial section, not a card.
- Header navigation order changed to: Products, Ethics, Artist, Contact.
- Global cards are flatter, lighter, and less visually heavy.
- Checkout, cart, contact, FAQ, policy, product detail, artist, and success pages were tightened to reduce oversized card blocks.

Design intent:

AZALI should feel breathable, tactile, and editorial, not like a loud template. The user should be able to scan the page comfortably on a computer screen without feeling surrounded by large boxes.
