# Customization sheet handoff

The owner mentioned a customization sheet, but it was not included in the generated project files. Place it here as:

`docs/azali-customization-options.xlsx`

Current implementation already supports database-driven customization groups and options:
- color select
- single select
- multi select
- text
- textarea
- boolean
- image swatch data fields
- price modifiers
- required/optional assignment through ProductCustomizationGroup

Seed data includes leather color, thread color, notebook paper type, and initials.
