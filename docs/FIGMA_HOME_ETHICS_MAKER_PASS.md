# AZALI, Figma homepage / ethics / maker pass

Applied after the uploaded `Untitled.fig` file.

Because the native `.fig` canvas is stored as Figma's binary `fig-kiwi` format, the implementation was based on the visible exported preview and the latest synced design direction, focused only on the three pages requested by the owner.

## Pages changed

### Home page
- Hero title changed to: `Objects shaped with patience, precision, and soul.` / `Des objets façonnés avec patience, précision et âme.`
- Hero body reframed around leather companions and ownership over time.
- CTAs simplified to `Shop pieces` and `+ The artist`.
- Category heading changed to `Start with the object.` / `Commencer par l’objet.`
- Featured heading softened to `Pieces ready to begin a longer life.`
- Hero visual system keeps the dark umber background, spiral watermark, and edited CTA hierarchy.

### Leather Ethics page
- Kept the manifesto title as the page title, not the menu label.
- Added a stronger editorial image band with caption.
- Split content into readable manifesto cards with alternating garnet/indigo rhythm.
- Closing section remains dark and declarative.

### Maker page
- Reworked into a lighter editorial layout instead of a heavy dark hero.
- First fold now follows the requested structure: image, italic quote, normal paragraph.
- Added a dark pull-quote section and alternating story sections below.

The rest of the site was intentionally left unchanged.
