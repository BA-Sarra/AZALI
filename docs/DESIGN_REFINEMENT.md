# AZALI Design Refinement

This update makes the interface calmer, more editorial, and less crowded.

## Palette behavior

- Parchment, sand, leather, and umber carry most of the interface.
- Garnet is used for desire and commerce: primary CTAs, prices, selected states, and key conversion accents.
- Indigo is used for identity and depth: secondary CTAs, language switcher, ethics/origin labels, craft links, and made-to-order markers.

## UI direction

- Removed the garnet glow/shadow from primary buttons.
- Reduced button size and made CTAs more matte.
- Increased section spacing and content breathing room.
- Reduced oversized text in page headings and cards.
- Reduced corner radius across core UI elements.
- Made product and category cards quieter, with subtle border hover states instead of heavy effects.

## Reading experience

Text blocks are narrower, body copy is smaller, and line-height is more relaxed. The site should feel like a premium artisan brand, not a dense e-commerce template.
